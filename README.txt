## Estructura de Ramas
El proyecto se desarrolló utilizando las siguientes ramas (branches):
- `main`: Rama principal de producción.
- `menu`: Estructura base del menú.
- `feature-fibonacci`: Implementación de la sucesión de Fibonacci.
- `feature-factorial`: Cálculo de factoriales.
- `feature-primos`: Verificación de números primos.
- `feature-perfectos`: Generación de números perfectos.
